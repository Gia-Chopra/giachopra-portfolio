# To-Do List App (Python)

## Overview
This is a simple **console-based To-Do List application** built in Python.  
It allows users to **add, edit, and manage tasks** along with their statuses. The app demonstrates Python fundamentals including lists, dictionaries, functions, loops, and user input handling.

---

## Features
- Add one or multiple tasks at a time
- Edit existing tasks
- Update task status:
  - Incomplete
  - In Progress
  - Completed
- Display all tasks in the current list

---

## Tech Stack
- Python 3.x
- Console-based interface
