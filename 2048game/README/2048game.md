# 2048 Game (Java)

## Overview
This is my implementation of the classic **2048 puzzle game** in Java.  
It demonstrates core object-oriented programming skills, including modular design, class interactions, and game logic implementation.

## Features
- Classic 2048 gameplay with a 4x4 board
- Tile merging and scoring system
- Undo last move feature
- Clean, modular code structure

## Tech Stack
- Java 17
- Object-Oriented Programming (OOP)
- Console-based UI
